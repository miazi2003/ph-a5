Question-1: What are the different ways to select an element in the DOM?

there are many methods to select elements in the DOM , some of them : getElementById() , getElementsByClassName() , getElementsByTagName() , querySelector() , querySelectorAll() , getElementsByName() , childNodes , firstChild / lastChild , children etc.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

"innerText" returns the visible text of an element, "innerHTML" returns the HTML markup inside an element, "textContent" returns the raw text content of an element

Question-3: What is event delegation in the DOM?

Event delegation is a technique in the DOM where a single event listener is attached to a parent element to handle events for its child elements.

Question-4: What is event bubbling in the DOM?

Event bubbling is a mechanism in the DOM where an event starts from the target element and propagates upward through its parent elements in the DOM tree.

Question-5: How do you create, add, and remove elements using JavaScript?

You can create elements with document.createElement(), add them using appendChild(), and remove them with removeChild() or parentElement.removeChild()
